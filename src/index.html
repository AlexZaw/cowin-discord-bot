About

Cowin Assist is a chat-bot that lets the user track the availability of covid vaccination centres in India via the CoWin API. It has a bunch of commands through which the user can set/update their desired filters to receive up to date information regarding the vaccine centres.


Features & Commands

  • Add the discord bot to a server
  • Register your district, pincode and age (just once, we'll save it for you)
  • Check for the availability of a vaccine center using the data provided
  • Navigate to COWIN site if slot available
  • Subscribe for hourly update on open slots if available

  • /check to manually check for available slots with the given parameters.
  • /help to show the list of available commands.
  • /notify-me to enable hourly notifications for the user if slot is avaiilable.
  • /set to set the user's age and district in the db.
